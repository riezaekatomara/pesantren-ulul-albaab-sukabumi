### ğŸ”§ Troubleshooting

<details>
<summary><strong>Port 5173 sudah digunakan</strong></summary>

Ubah port di `vite.config.js`:
```javascript

  server: {
    port: 3000 // Ganti dengan port yang diinginkan
  }
})
```
</details>

<details>
<summary><strong>Error saat npm install</strong></summary>

Coba hapus `node_modules` dan `package-lock.json`, lalu install ulang:
```bash
rm -rf node_modules package-lock.json
npm install
```
</details>

<details>
<summary><strong>Tailwind classes tidak muncul</strong></summary>

Pastikan `tailwind.config.js` sudah benar:
```javascript
content: [
  "./index.html",
  "./src/**/*.{js,ts,jsx,tsx}",
]
```
</details>

### ğŸ“¤ Deploy to GitHub Pages

1. Install gh-pages
```bash
npm install --save-dev gh-pages
```

2. Update `vite.config.js`
```javascript
export default defineConfig({
  base: '/pesantren-ulul-albaab-sukabumi/',
})
```

3. Add deploy script to `package.json`
```json
"scripts": {
  "deploy": "npm run build && gh-pages -d dist"
}
```

4. Deploy
```bash
npm run deploy
```
